# Assignment 1 PHP Enviroment

PHP environment that can be used for your assignment 1 web site. The code for the website will **still need to be submitted to Moodle**. Submissions cannot be via GitHub.

Details on the configuration and usage of the Codespace are in the [`codespaces.md` document](codespaces.md).

## Submitting to Moodle

To download your code from this repository, click the green **Code** button, **Local** tab, and select **Download ZIP**. Upload the downloaded zip file directly to the Moodle assignment submission area as instructed.
